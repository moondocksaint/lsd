## Why this example

This is the hybrid/visually structured case. The repository page carries meaning in prose, commands, code blocks, pipeline stage descriptions, plugin framing, and repository-style UI structure. It demonstrates when hybrid ingestion is preferable to pure text flattening.

## Key lesson

Repository-style pages are better treated as hybrid sources. The text layer is useful but does not fully capture the communicative structure of the page.
