## Why this example

This is the text-dominant control case. The page is prose-heavy, cleanly structured, and example-rich. It is a strong source for reviewer and rewriter skills and demonstrates when text-first ingestion is clearly the right choice.

## Key lesson

On prose-heavy, parser-friendly pages, text-first ingestion is primary and visual retrieval adds preservation confidence rather than extraction quality.
